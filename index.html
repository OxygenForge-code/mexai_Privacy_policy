MexAI ("biz", "bizim" veya "uygulama") olarak kullanıcılarımızın gizliliğine büyük önem veriyoruz. Bu Gizlilik Politikası, uygulamamızı kullanırken hangi bilgilerin toplandığını, nasıl kullanıldığını ve korunduğunu açıklar.

📊 Toplanan Bilgiler

1. Yerel Depolama (Local Storage)

  • AI bellek verileri (kullanıcı tercihleri, isim, projeler)
  • Sohbet geçmişi (sadece cihazda saklanır)
  • Ayarlar (tema, yazı boyutu, TTS tercihleri)

2. Dosya Sistemi Erişimi

  • /storage/emulated/0/MexAI/ klasöründe oluşturulan dosyalar
  • Kullanıcı tarafından oluşturulan HTML, JS, CSS dosyaları
  • Bu dosyalar sadece yerel cihazda saklanır, buluta gönderilmez

3. API İletişimi

  • Groq API'ye gönderilen mesajlar (geçici işlem için)
  • API yanıtları (gerçek zamanlı olarak işlenir, saklanmaz)

🛡️ Bilgi Kullanımı

Topladığımız bilgiler şu amaçlarla kullanılır:

  • ✅ AI asistanının kişiselleştirilmesi (bellek özelliği)
  • ✅ Kullanıcı tercihlerinin hatırlanması
  • ✅ Yerel dosya yönetimi ve düzenleme
  • ✅ Uygulama deneyiminin iyileştirilmesi
❌ Verileriniz asla:
• Üçüncü taraflarla paylaşılmaz
• Harici sunuculara gönderilmez
• Reklam amaçlı kullanılmaz
• Satılmaz veya ticari amaçla kullanılmaz

🔐 Veri Güvenliği

Tüm verileriniz sadece cihazınızda saklanır:

  • 📱 SharedPreferences: Ayarlar ve bellek verileri
  • 💾 Dahili Depolama: Sohbet geçmişi ve AI belleği
  • 📁 Dosya Sistemi: Oluşturulan dosyalar

Internete sadece API çağrıları için çıkılır ve bu çağrılarda sadece gönderdiğiniz mesajlar iletilir.

🔒 Overview

At MexAI ("we", "our", or "the app"), we take user privacy very seriously. This Privacy Policy explains what information is collected when you use our application, how it is used, and how it is protected.

By using MexAI, you agree to this policy. If you do not agree, please do not use the application.

📊 Information We Collect

1. Local Storage

  • AI memory data (user preferences, name, projects)
  • Chat history (stored only on your device)
  • Settings (theme, font size, TTS preferences)

2. File System Access

  • Files created in /storage/emulated/0/MexAI/ directory
  • User-created HTML, JS, CSS files
  • These files are stored locally only, not sent to cloud

3. API Communication

  • Messages sent to Groq API (for temporary processing)
  • API responses (processed in real-time, not stored)

🛡️ How We Use Information

The information we collect is used for:

  • ✅ Personalizing the AI assistant (memory feature)
  • ✅ Remembering user preferences
  • ✅ Local file management and editing
  • ✅ Improving application experience
❌ Your data is NEVER:
• Shared with third parties
• Sent to external servers (except API calls)
• Used for advertising
• Sold or used commercially

🔐 Data Security

All your data is stored only on your device:

  • 📱 SharedPreferences: Settings and memory data
  • 💾 Internal Storage: Chat history and AI memory
  • 📁 File System: Created files

Internet is only used for API calls, and only your messages are transmitted.

👤 Your Rights

You have the following rights:

  • 🗑️ Clear Memory: Settings > Clear All Memory
  • 💬 Clear Chats: Settings > Clear All Chats
  • 📁 File Management: Delete, edit, export files
  • ⚙️ Change Settings: Theme, language, TTS preferences

🤖 AI and Third Parties

MexAI uses the Groq API service:

  • Your messages are sent to and processed on Groq's servers
  • Groq's privacy policy: groq.com/privacy
  • MexAI is not responsible for Groq's data usage
